# Project Images

Place your project images in this folder. Supported formats: JPG, PNG, WebP

## File Naming Convention

- `mpaact.jpg` or `mpact.png` - MPact E-Commerce project
- `restaurant-billing.jpg` or `restaurant-billing.png` - Restaurant Billing System
- `hospital-management.jpg` or `hospital-management.png` - Hospital Management System
- `parallax-website.jpg` or `parallax-website.png` - Static Parallax Website

The images will be displayed in your portfolio's projects section. Recommend:
- Image dimensions: 800x600px or 16:9 aspect ratio
- File size: 100-300KB (optimized for web)
- Format: JPG for photographs, PNG for screenshots

After adding images, the portfolio will automatically display them in the project cards.
